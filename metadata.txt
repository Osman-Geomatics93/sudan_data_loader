    === VERSION 3.0: MAJOR RELEASE ===

    === NEW DATA SOURCES ===
    - OpenStreetMap/Overpass API Integration
      • Query POIs by category (hospitals, schools, roads)
      • Custom Overpass queries for Sudan
      • Real-time data fetching with caching

    - Sentinel Hub Satellite Imagery
      • Browse Sentinel-2 imagery for Sudan
      • Date range selection with cloud cover filter
      • NDVI, true color, false color composites

    - World Bank Development Indicators
      • Socio-economic indicators for Sudan
      • GDP, population, health, education metrics
      • Time series data visualization

    - NASA FIRMS Fire Data
      • Real-time fire/hotspot detection
      • Last 24h, 48h, 7 days options
      • Automatic Sudan boundary clipping

    - IOM Displacement Tracking
      • Internal displacement data
      • Population movement flows
      • DTM (Displacement Tracking Matrix) data

    === MODERN UI/DASHBOARD ===
    - Main Dashboard Panel with KPI cards
    - Interactive Charts (bar, pie, line)
    - Dark Mode Support
    - Modern Toast Notifications
    - Enhanced Faceted Search with fuzzy matching

    === QGIS DEEP INTEGRATION ===
    - Processing Provider with Sudan-specific algorithms
      • Clip by State
      • Buffer Analysis
      • Statistics Calculator
      • Join Attributes
    - Async Operations with QgsTask
    - Credential Manager Integration
    - Layer Tree Context Menu Integration
    - Custom Expression Functions
      • sudan_state_name(pcode)
      • sudan_area_km2()
      • sudan_is_darfur(state)
      • sudan_region(state)

    === AI/SMART FEATURES ===
    - Natural Language Query Parser
      • "Show all localities in Khartoum"
      • "Find health facilities near conflict"
    - Automated Report Generation with insights
    - Anomaly Detection (Z-score, IQR methods)
    - Predictive Analytics for conflict trends
    - Smart Data Recommendations

    === RESEARCH TOOLS ===
    - Citation Generator (APA, BibTeX, Chicago, MLA)
    - Data Provenance Tracking
    - Spatial Statistics (Moran's I, Getis-Ord Gi*)
    - Publication Export (journal templates)
    - Project Templates for workflows

    === EXISTING FEATURES ===
    - HDX Humanitarian Data Integration
    - ACLED Conflict Data Integration
    - Settings/Configuration Dialog
    - Layer Selection with remember option
    - Quick Labeling Tools (English/Arabic/P-Codes)
    - Style Preset Switcher
    - Data Info Panel
    - Search Panel with autocomplete
    - Bookmarks Panel (18 states + custom)
    - Statistics Panel with area calculations
    - Query Builder Dialog
    - Basemap Integration
    - Report Generation (HTML/PDF)
    - Export Dialog (multiple formats)
    - Sketching Toolbar
    - Processing Tools Integration
    - Data Validation Checker

    Data layers included:
    - Admin 0: Country boundary
    - Admin 1: States (18)
    - Admin 2: Localities
    - Administrative lines
    - Administrative points
