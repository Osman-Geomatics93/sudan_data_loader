    Features:
    - Load administrative boundaries (country, states, localities)
    - Administrative lines and points
    - Automatic styling with QML files
    - Download/Update data from GitHub releases
    - SHA256 verification for secure downloads
    - Offline support with bundled data

    Data layers included:
    - Admin 0: Country boundary
    - Admin 1: States
    - Admin 2: Localities
    - Administrative lines
    - Administrative points
